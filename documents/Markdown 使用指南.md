## 段落与行内样式

普通文字就是段落。行内可叠加：

这是 **粗体**、*斜体*、~~删除线~~，以及 `行内代码`。

也可以组合：**粗体里的 *斜体***。

快捷键（macOS 用 Cmd，Windows 用 Ctrl）：

- 粗体：`Mod + B`
- 斜体：`Mod + I`
- 删除线：`Mod + Shift + S`
- 行内代码：`Mod + E`

## 列表

行首输入 `- ` 或 `* ` 变成无序列表；输入 `1. ` 变成有序列表。`Tab` / `Shift + Tab` 可增减缩进。

- 无序一项
- 无序二项
  - 嵌套子项
  - 另一子项

1. 有序第一步
2. 有序第二步
   1. 嵌套步骤

快捷键：`Mod + Shift + L`（无序）、`Mod + Shift + O`（有序）。

## 引用

行首输入 `> `：

> 引用可以嵌套。
>
> > 内层引用

快捷键：`Mod + Shift + Q`。

## 代码块

行首输入三个反引号加空格（或 `Mod + Alt + C`）插入代码块，可写语言标识：

```ts
function greet(name: string): string {
  return `Hello, ${name}`;
}
```

## 链接与图片

链接写法：

[示例链接](https://example.com)

快捷键 `Mod + K` 可切换链接（新链接会先带占位地址，可再改）。

图片支持本地相对路径或远程 URL。把图片**粘贴**或**拖拽**进编辑器即可；本地文件会落到 `assets/images/`，正文保存相对路径，方便 Git 同步。

写法示例（把路径换成你的图片即可）：

`![说明文字](assets/images/example.png)`

## 常用输入技巧

- `# ` … `###### ` → 标题
- `- ` / `* ` / `1. ` → 列表
- `> ` → 引用
- 三个反引号 + 空格 → 代码块
- `---` → 分割线
- `**文字**` / `*文字*` / `~~文字~~` / 反引号包裹 → 行内样式
- `[文字](网址)` → 链接

其它：`Mod + S` 保存，`Mod + Z` / `Mod + Shift + Z` 撤销重做。内容以本地 `.md` 文件为准，可用任意编辑器打开。

## 暂未支持

以下语法会尽量保留原文、不丢内容，但编辑器里不会按富文本渲染：

- 表格（Table）
- 任务列表（`- [ ]`）
- 脚注、数学公式、Mermaid 等

写完可以把本文删掉或改成你自己的笔记——它只是空工作区的起步示例。
